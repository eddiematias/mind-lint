# /weekly-review — Weekly Retrospective

Review the week's knowledge activity with a mini-lint and content suggestions.

## Steps
1. **Learnings recap**: Group this week's learnings by category, show count
2. **Decisions recap**: List decisions made this week with status
3. **Error rules**: Show rules added this week
4. **Mini-lint**: Quick scan of wiki/ and memory/ for stale info or index drift
5. **Content ideas**: Analyze the week's learnings and decisions. Suggest 2-3 content ideas based on patterns. Offer to add to content/_pipeline.md.
6. **Skill suggestions**: If patterns emerge across multiple learnings, suggest extracting into a new skill file.
7. Report summary

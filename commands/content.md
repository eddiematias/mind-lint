# /content — Content Pipeline Manager

View and manage the content pipeline.

## Steps
1. Read content/_pipeline.md
2. Display status: ideas count, in-progress count, published count
3. Ask: "What do you want to do?"
   - **Capture new idea** → Create in content/ideas/ using template, update _pipeline.md
   - **Develop an idea** → Pick from Ideas, move to In Progress, create draft, load source material, start content-creation workflow
   - **Check on a draft** → Pick from In Progress, show status, offer to continue
   - **Publish** → Hand off to /publish
   - **Just browsing** → Done
4. Update _pipeline.md after any action

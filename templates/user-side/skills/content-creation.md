# Skill: Content Creation

## When to Load
Creating blog posts, LinkedIn posts, social threads, video scripts, newsletter content, or documentation about work being done.

## Workflow
1. Identify the key takeaway (one sentence)
2. Identify the audience and format
3. Pull source material from wiki/, memory/, and raw/
4. Create outline for approval
5. Draft
6. Review against format-specific guidelines
7. Final edit
8. Update content/_pipeline.md

## Format Guidelines

### Blog Post
- 800-2000 words
- Hook in first paragraph
- Headers every 2-3 paragraphs
- Code examples or screenshots where relevant
- End with a takeaway, not a generic conclusion
- Personal voice, not a textbook

### LinkedIn Post
- 1,300 characters max
- Hook in first line
- Personal angle or story
- 3-5 relevant hashtags

### Social Thread (X/Twitter)
- 5-10 tweets, each stands alone
- First tweet is the hook, last is the CTA
- Number them (1/7, 2/7, etc.)

### Video Script
- Talking points, not word-for-word
- Note screen recording moments
- Open with the problem, close with the result

### Notion Page (Client-Facing)
- Professional, clean
- Show process and thinking
- Include relevant visuals

## Rules
- Always also load @content/_pipeline.md
- Every piece traces to a source
- Update _pipeline.md when status changes
- The process of building things IS content

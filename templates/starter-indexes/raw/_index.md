# Raw Sources Index

## Daily Notes

Daily notes live in `journal/YYYY-MM/YYYY-MM-DD.md` and are part of the read-only substrate. They are NOT individually catalogued in this index — that would balloon the table and they are organized chronologically by their own folder structure. Agent commands (`/today`, `/close-day`, `/graduate`, future pattern-detection commands) read directly from `journal/`.
